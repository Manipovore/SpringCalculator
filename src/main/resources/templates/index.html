Calculatrice en ligne

Crée le 19/04/2019

Projet arthimétique simple codé en Java EE

L'algorithme du projet permet de récupérer une chaine de caractère, il analyse et traite en interne cette "String" de façon à renvoyer le résultat (Double|BigDecimal) après un processus d'opération arithmétique simple basé sur les opérateurs + - * / %.

Aussi, le coeur du processus prend en compte tous les nombres entiers, décimaux, naturels et relatifs. Le principe de priorité est respectée.

Technique Technologie utilisée

  • Java SE | Java EE -> Coeur du système | MVC
  • JS -> Simplement, affiche une calculatrice en lien avec le formulaire java
  • POO
  • RegEx
  • Css/Bootstrap
  • HTML5
